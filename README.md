# Modulus Fork by YGMaerz

This fork of [NVIDIA/modulus](https://github.com/NVIDIA/modulus) adds a **Darcy Flow** 
PDE pipeline with **AutoML** and **Active Learning** under:
`examples/cfd/darcy_autoML_active_learning/`.

See the [README in that subfolder](examples/cfd/darcy_autoML_active_learning/README.md) 
for details, or the 
[GETTING_STARTED.md](examples/cfd/darcy_autoML_active_learning/GETTING_STARTED.md)
which covers environment setup.
